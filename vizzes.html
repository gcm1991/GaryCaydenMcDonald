Data Visualization

Tableau

Tableau is my go-to tool for creating visuals and dashboards . I like to use Tableau to 'experiment' with different kinds of graphs and aesthetics or when I want my visual to have an interactive element.

R and ggplot2

R is an excellent visualization tool. I primarily use the ggplot2 package for my graphs. I like to use R when sharing code with colleagues is important or if I want all the stages of data analysis in one place.
